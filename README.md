Restaurant Reservation Website using Laravel

-   Vite 4.0.0
-   Tailwindcss 3.1.0

## Features

-   Admin panel
-   Categories
-   Menu Management
-   Table Management
-   Location Management
-   Make Reservation
-   Special Menu

```
git clone https://github.com/MeshackNzuki/ReservationMS.git
composer install
cp .env.example .env
php artisan key: generate
php artisan migrate
```

Then

```
php artisan serve
```

```
npm install
npm run dev
```
